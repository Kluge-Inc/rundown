Rundown is a simple RSS reader, based on periodic e-mail digests.

  • We use Ruby 2.1 and Rails 4.1.2
  • Complete documentation could be found in the Wiki
  • We track code comments and tasks with the help of watson-ruby
  • Code is documented in accordance with yard syntax

Docs generation:

gem install yard
yardoc --output-dir docs/api/ 'app/**/*.rb'
